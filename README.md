# 😁😁*项目使用须知*

# 😁*项目主要使用Vue+Element框架*

# 😁😁*此项目为PC端后台管理系统*

# 😁😁😁*使用时请先 npm install 安装扩展包*

# 😁*项目启动命令 npm run start*

# 😁*项目打包命令 npm run build*

### 😁*次项目有两个路口文件main.js(main-dev.js和main-prod.js)*

### 😁😁*开发时使用main-dev.js 使用本地的框架*

### 😁😁😁*打包时使用main-prod.js 将项目中框架等等不打包,使用外链cdn方式*
